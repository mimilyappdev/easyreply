## サービス概要

- **サービス名**：EasyReply（ラクリプ）
- **コンセプト**：返信判断回数を減らす「返信フロー半自動化UI」

## サンドイッチ構造（FIX）

### 受ける・受ける詳細
```
上固定：
  この度はお問合せいただきありがとうございます。
  SeventhWorksです。

差し込みゾーン（ボタンで選択・複数可）：
  受ける       → 受諾・見積案内 / 見積もり提示
  受ける詳細   → 予算確認 / 納期確認 / 必要事項テンプレ

下固定：
  どうぞよろしくお願い申し上げます。
```

### 断る
```
上固定：
  この度はお問い合わせいただきありがとうございます。
  せっかくお問合せいただいたにも関わらず今回依頼をお受けすることが出来かねます。

差し込みゾーン（ボタンで選択・複数可）：
  予算理由 / 納期理由 / スケジュール理由 / その他（自由入力）

下固定：
  また機会がございましたらお問い合わせいただけますと幸いです。
  何卒よろしくお願い申し上げます。
```

## フェーズ・テンプレート構成

### 依頼前
```
受ける：
  差し込み → 受諾・見積案内 / 見積もり提示

受ける（詳細記載）：
  差し込み → 予算確認 / 納期確認 / 必要事項テンプレ

断る：
  差し込み → 予算理由 / 納期理由 / スケジュール理由 / その他
```

### 依頼中（4セクション）
```
振込後着手：制作開始連絡
ラフ：ラフ確認 / ラフ確認後返信・修正あり / ラフ確認・修正点なし
修正：ラフ修正 / 修正確認・問題なし
制作物・振込：制作物確認 / 制作完了・振込依頼（変数：bank_info）
```

### 完了
```
納品＆完了連絡（振込なし）（変数：file_url / password）
納品＆完了連絡（振込あり）（変数：file_url / password）
```

## 変数（{{key}}形式）

```
{{bank_info}}  → 振込先情報（セッション保持のみ・localStorage不使用）
{{file_url}}   → ダウンロードURL（同上）
{{password}}   → パスワード（同上）
```
