JSON 構造のガイド

以下の構造でテンプレートを管理します。templates 内の text を編集することで文言をカスタマイズできます。

{
  "creator": {
    "name": "屋号・クリエイター名",
    "twitter": "https://twitter.com/...",
    "web": "https://..."
  },
  "sandwiches": {
    "accept": { "top": "挨拶文...", "bottom": "締め文..." },
    "decline": { "top": "挨拶文...", "bottom": "締め文..." }
  },
  "templates": {
    "btn_accept_guide": {
      "label": "受諾・見積案内",
      "type": "static",
      "text": "カスタマイズしたテキスト..."
    }
  }
}

※ index.html にエクスポート機能が追加されると、現在の設定をここに読み込めます。
