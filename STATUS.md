最終更新: 2026-05-20

## 現在の仕様

### アプリ概要
フリーランスクリエイター向けの返信テンプレ管理ツール。
登録不要・ブラウザ完結・1ファイル完結構成。

### ファイル構成
```
/
├── index.html          # LP（公開中）
├── privacy.html        # プライバシーポリシー
├── terms.html          # 利用規約
└── app/
    ├── index.html      # 「準備中」ページ（noindex設定済み）
    ├── my.html         # 個人テンプレ入り開発用アプリ（非公開URL）
    └── settings.html   # 設定ページ（テーマ切替 Warm/Onyx 実装済み）
```

### 現在実装されている機能
- プリセットフェーズ・テンプレの表示と使用
- 差し込み変数（`{{price}}` など）
- 敬語トーン切替
- テーマ切替（Warm / Onyx）※ settings.html
- お気に入り保存ボタン（UIのみ、ロジック未実装）
