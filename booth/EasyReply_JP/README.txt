【使い方】
----------------------------------------------------------------

EasyReplyはブラウザで動くアプリです。
HTMLファイルをブラウザで開くだけで使えます。インストール不要です。
データはすべてお使いのデバイス内（ブラウザのローカルストレージ）に保存されます。
データは端末ごとに保存され、自動同期されません。

■ 起動方法

ダウンロードした「EasyReply_JP.html」をブラウザ（Chrome / Safari / Edge 等）
で開いてください。ホーム画面に追加するとアプリのように使えます。

iPhone：Safariで開いて「共有 → ホーム画面に追加」
Android：ブラウザメニューから「アプリをインストール」または「ホーム画面に追加」

■ データの保存について（重要）

本アプリのデータはブラウザのローカルストレージに保存されます。
以下の操作でデータが消える場合があります：
・ブラウザのキャッシュ削除・サイトデータ削除
・ホーム画面からアイコンを削除
・機種変更・端末初期化

定期的に「設定 → エクスポート」でJSONバックアップを取ることを
強くおすすめします。

■ 基本の使い方

1. フェーズを選ぶ
   画面上部のタブから「依頼前」「依頼中」「完了」を選びます。

2. 返答パターンを選ぶ
   「受ける」「断る」などのチップから、状況に合うものを選びます。

3. ボタンを選んで文章を組み立てる
   左パネルのボタンをタップすると、右のプレビューに文章が表示されます。
   複数のボタンを選ぶと積み上げて表示されます。

4. コピーして使う
   「コピー」ボタンでクリップボードにコピーできます。

5. DLリンクを添付する
   「DLリンク」ボタンからダウンロードURLとパスワードを入力すると、
   プレビューの末尾に自動で追加されます。

6. データをバックアップする
   設定画面（⚙）の「エクスポート」からJSONファイルとして保存できます。
   機種変更のときは「インポート」で復元してください。

■ テーマカラーについて

設定（⚙）→ テーマカラーから、デフォルト・ダーク・マイカラー（2種類）
を切り替えられます。マイカラーは5色を自由にカスタマイズできます。

■ 対応環境

Chrome / Safari / Edge 等の主要なモダンブラウザ最新版を推奨します。
Firefoxおよび一部特殊ブラウザでは正常に動作しない場合があります。

■ アップデートについて

不具合修正や機能改善を行った場合、Boothのダウンロードページから最新版を取得できます。
ただし、将来的な機能追加・仕様変更・新製品への無償アップデートを保証するものではありません。
