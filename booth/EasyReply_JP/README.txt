## 【使い方】

EasyReplyはブラウザで動くアプリです。

HTMLファイルをブラウザで開くだけで利用できます。
インストールは不要です。

データはすべてお使いのデバイス内
（ブラウザのローカルストレージ）に保存されます。

データは端末ごとに保存され、自動同期は行われません。

■ 起動方法

ダウンロードした「EasyReply_JP.html」を
ブラウザ（Chrome / Safari / Edge 等）で開いてください。

ホーム画面に追加すると、アプリのように利用できます。

iPhone：
Safariで開く → 「共有」→「ホーム画面に追加」

Android：
ブラウザメニュー → 「アプリをインストール」
または「ホーム画面に追加」

■ データの保存について（重要）

本アプリのデータはブラウザのローカルストレージに保存されます。

以下の操作などにより、データが消失する場合があります。

・ブラウザのキャッシュ削除・サイトデータ削除
・ホーム画面からの削除や再インストール
・機種変更・端末初期化

定期的に「設定 → エクスポート」で
JSONバックアップを取得することを強くおすすめします。

■ 基本の使い方

1. フェーズを選ぶ

画面上部のタブから「依頼前」「依頼中」「完了」を選びます。

2. 返答パターンを選ぶ

「受ける」「断る」などのチップから、
状況に合うものをタップします。

3. ボタンを選んで文章を組み立てる

左パネルのボタンをタップすると、
右のプレビューに文章が表示されます。

複数のボタンを選ぶと積み上げて表示されます。

4. プレビューを編集する（任意）

プレビューエリアは直接編集できます。
宛名や細かい内容を書き足してから送ってください。

5. コピーして使う

「コピー」ボタンでクリップボードにコピーできます。

6. DLリンクを添付する（任意）

「DLリンク」ボタンからダウンロードURLと
パスワードを入力すると、プレビューの末尾に
自動で追加されます。

7. データをバックアップする

設定画面（⚙）の「エクスポート」から
JSONファイルとして保存できます。

機種変更時は「インポート」から復元してください。

■ テーマカラーについて

設定（⚙）→ テーマカラーから
デフォルト・ダーク・マイカラー（2種類）を切り替えられます。

マイカラーは5色を自由にカスタマイズできます。

■ 対応環境

Chrome / Safari / Edge 等の
主要なモダンブラウザ最新版を推奨します。

Firefoxおよび一部特殊ブラウザでは
正常に動作しない場合があります。

■ アップデートについて

不具合修正や機能改善を行った場合、
Boothのダウンロードページから最新版を取得できます。

ただし、将来的な機能追加・仕様変更・
新製品への無償アップデートを保証するものではありません。

## 【無料版からの移行手順】

1. 無料版で「設定 → エクスポート」を実行し、
   バックアップを保存します

2. ダウンロードしたEasyReply_JP.htmlを開きます

3. ホーム画面に追加していた場合は、
   無料版のショートカットを削除し、
   改めて追加してください

4. 「設定 → インポート」でバックアップを復元してください
